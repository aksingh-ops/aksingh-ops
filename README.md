### ✈️ AA Crew Absence Prediction & Reserve Optimization
> `Aviation` · End-to-end ML system predicting crew sick-calls and optimizing reserve staffing

**Stack:** Python · XGBoost · BTS Flight Data · SHAP

- Reserve cost optimization layer estimated **$1.3M annual net benefit** across DFW and CLT hubs

### 👥 Employee Attrition Risk Scoring
> `HR & People Analytics` · Individual-level 90-day attrition prediction with SHAP explainability

**Stack:** Python · Random Forest · IBM HR Dataset · SHAP

- Predicted voluntary attrition on **1,470 IBM HR records** — AUC **0.748**, Recall **0.511**
- Resolved 5.2:1 class imbalance with SMOTE; validated High-risk tier at **41.7% actual attrition vs 8.1% Low-risk**
- SHAP TreeExplainer surfaced top drivers per employee (StockOptionLevel, JobLevel, JobSatisfaction)

[→ View Repository](https://github.com/aksingh-ops/employee-attrition-risk-scoring)

### ⚖️ Pay Equity Anomaly Detection Pipeline
> `HR & People Analytics` · Automated Alteryx ETL + OLS regression detecting statistically significant pay disparities

**Stack:** Python · Alteryx ETL · OLS Regression · Statsmodels · Tableau

- Joined 3 HR source systems (HRIS, Comp, Performance) via **13-tool Alteryx pipeline** with zero manual steps
- OLS regression on **950 employees** achieved R-squared **0.9325** — controlled gender gap **2.52% at p=0.0005**
- Detected **4 of 5 injected pay gaps** including a Finance VP cohort of only 15 employees (p=0.0095)

[→ View Repository](https://github.com/aksingh-ops/pay-equity-anomaly-detection)
